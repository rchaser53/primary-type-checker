# how to install

```js
$ yarn add primary-type-checker  # or npm install primary-type-checker
```

# how to play

```js
$ pmc filePath        # ex. pmc index.js
```

## support case

- assignment
- declaration
- if condition
- for statement
- while statement
- nest block
